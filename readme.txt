This program provides a basic Ipod emulator called UtPod. The packaged code tests the implementation of this
UtPod to make sure it's functionality is sound.

To compile: type  'make' (without quotation marks) into the console 

To run: type  './test ', without quotation marks; the program will automatically run a driver to 
		test the functionality of the UtPod

Preconditions: None. Make sure the file names are not changed so that the makefile will run successfuly
	
Postcondition: Program will display the results of each test on the screen, with a short explanation for each.
